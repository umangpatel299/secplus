## 🌐 Hosting on GitHub Pages

1. Create a new GitHub repository
2. Upload all files from this zip
3. Go to **Settings → Pages**
4. Set source to **main branch / root folder**
5. Your site will be live at `https://<username>.github.io/<repo>/`

## 📁 File Structure
All files are self-contained HTML — no build step, no dependencies, no server required.
The `index.html` is the homepage (Study Hub) that links to all individual guides.
